typedef void(* smCB_t) (uint8 *secretKey, uint8 *publicKeyX, uint8 *publicKeyY)

Application callback to the SM for ECC keys and Diffie-Hellman Shared Secret (ECDH).

Parameters
secretKeyPrivate key when ECC keys were requested. ECDH shared secret when DHKey is requested.
publicKeyXLocal Public Key X-Coordinate. Not used when DHKey is returned.
publicKeyYLocal Public Key Y-Coordinate when ECC keys were requested. Not used when DHKey is returned.
