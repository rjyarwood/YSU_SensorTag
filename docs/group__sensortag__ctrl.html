static void SensorTag_init ( void  )
static

Initializes the SensorTag and its BLE configurations.

Parameters
none
Returns
none

This function initializes BLE and any board drivers. This includes registering the entire app with the ICALL dispatcher, creating a message queue for RTOS messages. It also initializes a clock for periodic events and initializes the GAP and GATT. This is where you call any functions that register services.
